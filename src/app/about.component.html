BiaSearch Mission and Goal

it is sometimes hard to know whether or not the news you are receiving is impartial. In fact, every source owns an innate bias upon creaton. Some are certainly more extreme than others, but regardless, one should not limit themselves to one source. And so, we created BiaSearch with the goal of broadening the spectrum of news.

How does it work?

The front end of the application was created using Angular 2. The site is hosted on AWS.
